# HLTVEventParser

Parser for get teams and players stat by event and export it to google sheets and csv files

## Setup and run

1. CLone project: 
    ```bash
    git clone https://github.com/FlakeV/HLTVEventParser.git
    ```
2. Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
3. Запустите проект:
    ```bash
    python3 main.py
    ```
